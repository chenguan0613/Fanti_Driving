# Fanti Driving

## Development

0. Requirements:
   - Python >= `3.13`

1. Create virtual environment:

   ```bash
   python -m venv .venv
   ```

2. Enter the virtual environment:

   ```bash
   source .venv/bin/activate # Linux
   .\.venv\bin\Activate.ps1 # Windows
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
